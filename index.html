카카오와 함께하는
제주 Web Conference

오는 6월 3일 제주시 JDC 챌린저 홀에서 ‘카카오와 함께하는 제주 Web Conference가 개최된다. 이번 컨퍼런스는 카카오, (주)위니브와 멋쟁이 사자처럼 주관 하에 WEB 기술에 관련된 국내외 커뮤니티, 제주 커뮤니티와의 교류를 통하여 지리적인 한계를 극복하고 기술 발전에 대한 다양한 시각을 나누는 장을 마련하기 위해 개최된다. 또한 국내외 최고 기술자들과 제주 개발자의 밋업을 통해 제주의 웹 기술 역량강화 도모를 할 수 있다.

컨퍼런스는 'Part 1. 차세데 웹기술'과 'Part 2. 주니어 개발자 토크콘서트' 총 2개 세션이 진행되며, 주니어 개발자와 시니어 개발자 등이 대거 참가해 향후 FE기술과 WEB 발전을 위한 발표가 진행된다. 컨퍼런스 참여 희망 시 제주 웹 컨퍼런스 공식 홈페이지를 통해 사전 등록할 수 있으며 온라인 참석자를 위해 실시간 스트리밍 플랫폼도 제공한다.
